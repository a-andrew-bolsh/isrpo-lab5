# Модули геометрических вычислений

##  Модуль `circle.py`

**Назначение**: Библиотека для работы с геометрическими параметрами окружности

###  `area(r)`
**Назначение**: Вычисляет площадь окружности по заданному радиусу

**Параметры**:
- `r` (float) - радиус окружности

**Возвращаемое значение**:
```
math.pi * r * r - площадь круга ($S = \pi r^2$)
```

### `perimeter(r)`
**Назначение**: Вычисляет длину окружности по заданному радиусу

**Параметры**:
- `r` (float) - радиус окружности

**Возвращаемое значение**:
```
2 * math.pi * r - длина окружности ($P = 2\pi r$)
```

## Модуль `triangle.py`

**Назначение**: Библиотека для работы с геометрическими параметрами треугольника

### `area(a, b, c)`
**Назначение**: Вычисляет площадь треугольника, используя формулу Герона

**Параметры**:
- `a` (float) - первая сторона
- `b` (float) - вторая сторона
- `c` (float) - третья сторона

**Возвращаемое значение**:
```
sqrt(p * (p-a) * (p-b) * (p-c)) - площадь треугольника
```

**Примечание**:
- Использует вспомогательную переменную `p` (полупериметр)
- Предполагает, что стороны образуют существующий треугольник

###  `perimeter(a, b, c)`
**Назначение**: Вычисляет периметр треугольника (сумма длин всех сторон)

**Параметры**:
- `a` (float) - первая сторона
- `b` (float) - вторая сторона
- `c` (float) - третья сторона

**Возвращаемое значение**:
```
a + b + c - периметр треугольника
```

## Модуль `rectangle.py`

**Назначение**: Библиотека для работы с геометрическими параметрами прямоугольника

### `area(a, b)`
**Назначение**: Вычисляет площадь прямоугольника

**Параметры**:
- `a` (float) - первая сторона
- `b` (float) - вторая сторона

**Возвращаемое значение**:
```
a * b - площадь прямоугольника
```

###  `perimeter(a, b)`
**Назначение**: Вычисляет периметр прямоугольника

**Параметры**:
- `a` (float) - первая сторона
- `b` (float) - вторая сторона

**Возвращаемое значение**:
```
2 * (a + b) - периметр прямоугольника
```

##  Модуль `square.py`

**Назначение**: Библиотека для работы с геометрическими параметрами квадрата

###  `area(a)`
**Назначение**: Вычисляет площадь квадрата

**Параметры**:
- `a` (float) - сторона квадрата

**Возвращаемое значение**:
```
a * a - площадь квадрата
```

###  `perimeter(a)`
**Назначение**: Вычисляет периметр квадрата

**Параметры**:
- `a` (float) - сторона квадрата

**Возвращаемое значение**:

```
4 * a - периметр квадрата
```
